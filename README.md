# Cosmie

Space information web application using NASA API.

## About the Project 

Cosmie is a web application that displays space data using NASA Open APIs.
The application allows users to view different types of space information
in one dashboard, including Astronomy Picture of the Day, Mars weather,
Near Earth Objects, solar activity events and NASA image search.

## Features

- User registration and login
- Dashboard with space data
- Astronomy Picture of the Day
- Mars Weather information
- Near Earth Objects data
- Solar activity events (DONKI)
- NASA image search
- Customizable dashboard

## Technologies used

- Python
- Flask
- NASA Open API
- Supabase PostgreSQL
- HTML
- CSS
- Render (deployment)

## Installation

1. Clone the repository

git clone https://github.com/zlatadaskevica/cosmie

2. Navigate to the project folder

cd cosmie

3. Create virtual environment

python -m venv venv

4. Activate virtual environment

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

5. Install dependencies

pip install -r requirements.txt

6. Run the application

python main.py

## API

This project uses NASA Open APIs:
- APOD (Astronomy Picture of the Day)
- Mars Weather
- Near Earth Objects
- DONKI (solar activity)
- NASA Image Library
